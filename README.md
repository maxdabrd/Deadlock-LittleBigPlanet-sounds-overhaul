# Deadlock-LittleBigPlanet-sounds-overhaul
Added around 70 new LittleBigPlanet sounds, overhauling a big part of the sound environment of Deadlock. All sounds are sourced and mixed from LittleBigPlanet 1/2's source material, all copyrights owned by Media Molecule.

Includes:

- Dynamic main menu music
- 22 shop music
- Buy menu sounds
- Mid boss sounds
- Soul orbs sounds
- Box and statue sounds
- Urn carry music
- Slot machine music and sounds
- Debuff/jump pad sounds
- Spawning noise
- Guardian / Walker / Patron / Base stingers
- Party join / leave sounds
- Commend give / receive sounds
- Punch sounds (addon, not included in main bundle)
- Etc

<<<IMPORTANT>>>
The Main Bundle includes every module inside it EXCEPT the Punch sound module.

For nerd reasons, the Mid Boss module does NOT work with the Shop Music module or with the Soul Orbs / Boxes & Statue buffs module (Shop Music and Soul Orbs work together fine though). 

DO NOT have the Mid Boss module installed if you have one or the other incompatible ones, some sounds won't have the right volume.
