REMINDER
-----------------------
the Mid Boss module does NOT work with the Shop Music module or with the Soul Orbs / Boxes & Statue buffs module (Shop Music and Soul Orbs work together fine though). 

DO NOT have the Mid Boss module installed if you have one or the other incompatible ones, some sounds won't have the right volume.
